## Setup Guides

### NewsBot (Recommended for most users)
1. Visit [newsbot.app](https://newsbot.app)
2. Click "Add to Discord" and authorize
3. Configure feeds via web dashboard or `/newsbot` commands
4. Set up news channels in your server

**Time to setup:** <5 minutes

### RSS-Bridge (Recommended for technical users)
1. Install RSS-Bridge on your server: `docker run -d --name rss-bridge -p 80:80 rssbridge/rss-bridge`
2. Clone discord bot: `git clone https://github.com/RSS-Bridge/discord-bot.git`
3. Configure feeds in `config.ini`
4. Run bot: `python bot.py`
5. Invite to Discord via OAuth2

**Time to setup:** 15-30 minutes (depending on hosting)


### Reddit-News-Bot (Recommended for Reddit communities)
1. Visit [top.gg](https://top.gg) and search "Reddit-News-Bot"
2. Click "Invite" and authorize
3. Configure subreddits via `/reddit` commands
4. Set up notification channels

**Time to setup:** <5 minutes
