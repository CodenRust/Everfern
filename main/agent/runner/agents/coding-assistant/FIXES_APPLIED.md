# AI Coding Assistant - Fixes Applied

## Summary

All hardcoded framework detection and manual signals have been removed. The AI now learns exclusively from the actual codebase structure without any manual pattern matching.

## Changes Made

### 1. **Removed Hardcoded Framework Detection** ✅

**Before (coding-specialist.ts):**
```typescript
// ❌ Hardcoded patterns
if (allFiles.some(f => f.includes('next.config') || f.includes('pages/'))) {
  projectStructure.framework = 'nextjs';
} else if (allFiles.some(f => f.includes('src/App.tsx') || f.includes('src/App.jsx'))) {
  projectStructure.framework = 'react';
} else if (allFiles.some(f => f.includes('express') || f.includes('app.js'))) {
  projectStructure.framework = 'express';
}
```

**After (coding-specialist.ts):**
```typescript
// ✅ AI learns from actual codebase

  language: 'unknown', // Will be detected by AI from actual code
  dependencies: [],
  framework: undefined, // Will be detected by AI from actual code
  testFramework: undefined // Will be detected by AI from actual code
};
```
